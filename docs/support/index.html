Common Questions

Does the app send emails automatically?
The app opens the iOS system mail composer with a prepared message. The user chooses whether to send it.

Does the app require an account?
No. Settings and history are stored locally on the device.

Can family members be alerted when a check-in is missed?
Not in version 1.0.0. That feature requires a server or external delivery service.
