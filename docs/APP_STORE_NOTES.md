## Privacy Summary

The app stores profile, reminder, and check-in history only on the user's device. The app does not create accounts, does not run its own backend, and does not directly send data to a server. Check-in emails are sent by the user through the iOS system mail interface.

## Review Notes Draft

This app is designed as an offline-first family safety check-in utility. It does not create accounts or use an app-owned backend. When the user taps the check-in button, the app presents the iOS system mail composer with a prefilled family email, subject, and body. The user remains in control of sending the email.
